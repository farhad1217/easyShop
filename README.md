# EasyShop - বাজার সহায়ক

ফ্যামিলি বাজারের লিস্ট জমা দেবে এবং ম্যানেজমেন্ট ঠিকানায় পৌঁছে দিবে।

## বৈশিষ্ট্য

### ফ্যামিলি সেকশন
- রেজিষ্ট্রেশন ও লগইন
- বাজারের লিস্ট জমা দেওয়া (আইটেম ও পরিমাণসহ)
- লিস্টের স্ট্যাটাস দেখা (পেন্ডিং, যাচাইকৃত, পৌঁছানো)

### ম্যানেজমেন্ট সেকশন
- এডমিন লগইন
- সব ফ্যামিলির লিস্ট দেখা (তারিখ ও সময়সহ)
- ঠিকানা অনুযায়ী লিস্ট যাচাই করা
- পৌঁছানোর পর স্ট্যাটাস আপডেট করা

## চালানো

```bash
# মাইগ্রেশন
python manage.py migrate

# এডমিন একাউন্ট তৈরি (প্রথমবার)
python manage.py create_admin
# Username: admin, Password: admin123

# সার্ভার চালু
python manage.py runserver
```

ব্রাউজারে যান: http://127.0.0.1:8000/
